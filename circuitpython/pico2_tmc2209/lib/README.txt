Adafruit CircuitPython helpers. Included to ease installation. Master copies are maintained via Adafruit's CircuitPython website not via this project.
These are included to support position sensors
- lis3dh (accelerometer)
- as5600 (hall effect rotation sensor)

See https://github.com/adafruit/Adafruit_CircuitPython_Bundle
See https://learn.adafruit.com/welcome-to-circuitpython/circuitpython-libraries
See https://circuitpython.org/libraries

These may need to be updated from the original source as CircuitPython versions appear in the future.
These copies are initially for CircuitPython 9.2 support.
