This folder contains circuitpython code for the Pilomar project.

The tiny2040 subfolder contains the code.py program on the following setup:
    Pimoroni Tiny2040 microcontroller. 
    CircuitPython 7.2 to 8.2 only.
    drv8825 stepper motor drivers only.

The tiny2350 subfolder contains the code.py program on the following setup:
    Pimoroni Tiny2350 microcontroller. 
    CircuitPython 9.2 or later with RP2350 support.
    drv8825 stepper motor drivers only.

The pico2_tmc2209 subfolder contains the code.py program on the following setup:
    Raspberry Pi Pico2 microcontroller.
    CircuitPython 9.2 or later with Pico2 RP2350 support.
    tmc2209 stepper motor drivers only.
